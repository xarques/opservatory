Infrastructure languages are essential to manage andÂ automate provisioning of Cloud resources. But they are neither easy to learn nor easy to test.

Building OPServatory
--------------------

```bash
git clone git@github.com:xarques/opservatory.git
cd opservatory
bundle install
yarn install
rails db:create db:migrate db:seed
rails server
```
